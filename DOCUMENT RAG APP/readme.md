# 📄 Document RAG App

This submodule contains the **frontend and backend** implementation for the Document Q&A chatbot system,

Users can register, log in, upload documents, and ask intelligent questions based on the content of the documents using Retrieval-Augmented Generation (RAG).

---

## 📁 Folder Structure

- **client/** – React frontend for user interaction
- **server/** – Used node js

---

## ✨ Features Implemented by Me

- ✅ User Registration
- ✅ User Login with basic validation
- ✅ Document Upload (PDF/DOCX)
- ✅ Storage and preprocessing of uploaded documents
- ✅ UI integration with backend via API
- 🔄 In Progress: Document-based question answering using RAG

---

## ⚙️ Tech Stack

| Component          | Technology                                       |
| ------------------ | ------------------------------------------------ |
| Frontend           | React.js                                         |
| Backend            | FastAPI (Python) / node js                       |
| Auth Handling      | Custom JWT-based Auth (or local/session storage) |
| Document Parsing   | Python (PDFMiner / PyMuPDF / DOCX)               |
| Embedding & Search | Langchain, HuggingFace, FAISS                    |
